# AI Usage

I'm writing this in 2026. Of course I'm going to be using Large Language Models as a part of what I do. However, I want to be clear and open about this. I will be using "AI" as I work on these projects. However, I will be doing all the writing, along with any contributors. I will create the lessons. I will write the code. I will then use AI to validate the technical aspects of the code, do some light copy editing, and as a sounding board to ensure my lessons and plans cover everything necessary to a sufficient detail.

In short, this will not be AI generated. Period.
